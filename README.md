Kushagra Singh, Indira Sen, Ponnurangam Kumaraguru <br />
ACL SRW, 2018 <br />
<br />

(ii)  Language identification tool (English, Hindi, Rest)<br />
(iii) Named Entity Recogntion tool (Person, Location, Organisation)<br />
<br />
